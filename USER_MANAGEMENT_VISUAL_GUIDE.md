## Interactive States

### Buttons
- **Default**: Solid background with shadow
- **Hover**: Slight lift (translateY -2px) + stronger shadow
- **Active**: Pressed effect (translateY 0px)
- **Disabled**: 50% opacity, no interaction

### Cards
- **Default**: Subtle shadow
- **Hover**: Lift effect + stronger shadow
- **Active**: Scaled slightly (transform: scale(1.02))

### Inputs
- **Default**: Border #e5e7eb
- **Focus**: Border #3b82f6 with blue glow
- **Error**: Border #ef4444 with red glow
- **Disabled**: Background #f3f4f6, text gray

### Modal
- **Overlay**: Black with 50% opacity
- **Content**: White with large shadow
- **Animation**: Fade in + slide up (300ms)

### Fade In
```css

  from { opacity: 0; }
  to { opacity: 1; }
}

```

### Slide Up
```css
@keyframes slideUp {
  from { 
    transform: translateY(20px);
    opacity: 0;
  }
  to { 
    transform: translateY(0);
    opacity: 1;
  }
}
Duration: 300ms
```

```css
transform: translateY(-2px);
transition: all 0.2s ease;
```
